# Meu Primeiro Projeto Docker na Nuvem (AWS EC2)

Este repositório faz parte dos meus estudos práticos de Infraestrutura, Cloud e DevOps.

## 🛠️ O que foi feito:
* **Infraestrutura:** Criação e configuração de uma instância EC2 (Ubuntu) na AWS.
* **Segurança de Rede:** Configuração de Security Groups para gerenciamento de portas (80, 8080, 8081).
* **Containers:** Instalação do Docker e criação de um `Dockerfile` customizado.
* **Servidor Web:** Criação de uma imagem personalizada utilizando o Nginx para rodar uma aplicação web isolada.
* **Controle de Versão:** Gerenciamento do código e deploy utilizando Git e chaves criptografadas SSH.
